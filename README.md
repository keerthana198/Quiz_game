# CodeMaster – Programming Quiz Game
Programming Quiz Game using HTML, CSS, JavaScript

## Overview
CodeMaster is a simple web-based programming quiz game built using HTML, CSS, and JavaScript.
The game allows users to answer multiple-choice programming questions and test their knowledge in an interactive way.
The quiz includes a timer, score calculation, and navigation controls to create a better learning experience.

## Features
- Multiple-choice programming questions
- Timer for each question
- Score tracking
- Simple and user-friendly interface

## Technologies Used 
- HTML – Structure of the quiz interface
- CSS – Styling and layout of the game
- JavaScript – Quiz logic, timer, and answer validation
